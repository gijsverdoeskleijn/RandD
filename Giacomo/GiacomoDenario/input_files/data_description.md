
Review the attached A&A paper titled "Precision astrometry with Euclid VIS: a transferable focal-plane correction from Gaia-CRF3".
The paper discusses the astrometric performance of Euclid Q1 VIS single calibrated-frame catalogues by comparing source positions with the inertial optical reference frame defined by Gaia-CRF3.
It presents a detector-space distortion model (B26) that corrects geometric distortions in Euclid VIS astrometry.
The paper is organized into sections describing the data, methods, validation, and conclusions.
